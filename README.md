# JacksFirstPCB
I made a Simon says game with a PCB And tutorial from pathfinder, I wanted to do this because i needed a good easy first project to knock out, the game i made reflects my love for the simon game that i got for christmas when I was much younger. This project will help reduce boredom and also give people someting to fidget with and play.
